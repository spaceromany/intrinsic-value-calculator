# Intrinsic Value Calculator

한국 주식 내재가치 분석 웹앱 (Flask + 네이버 금융 크롤링)

## 이론적 근거 (《현명한 투자자 해제》 요약)

### 안전마진 (Margin of Safety)
- 내재가치(V)와 주가(P)의 차이. V-P가 클수록 안전
- 비율로 측정: `안전마진(%) = (V - P) / P × 100`
- 3중 안전장치: (1) 기업 수익성 자체의 안전마진 (2) 내재가치 대비 저가 매수 (3) 투자자 실수에 대한 완충
- "안전마진이 충분하다면 실적 악화의 충격을 흡수하는 완충재(Cushion) 역할"

### RIM (Residual Income Model, 초과이익모형)
- 그레이엄의 안전마진 개념을 현대적으로 표현한 가치평가 기법
- **공식**: `V = BPS + (ROE - r) × BPS / (r - g)`
  - V: 내재가치, r: 주주 요구수익률, g: 장기성장률
  - ROE > r 이면 초과이익 존재 → V > BPS (프리미엄)
  - ROE < r 이면 V < BPS (디스카운트)
- **적정PBR**: `V/BPS = (ROE - g) / (r - g)`

### 본 프로젝트의 내재가치 계산 (간편법)
- **EPS 가중평균** = (최근년도 EPS × 3 + 전년도 EPS × 2 + 전전년도 EPS × 1) / 6
- **내재가치** = (EPS 가중평균 × 10 + 최근년도 BPS) / 2
- **자사주 조정**: 내재가치 × (100 / (100 - 자사주비율))
  - 자사주가 없으면 조정 안 함

### PER과 적정주가 (2장)
- PER = 주가 / EPS (투자원금 회수기간)
- **경상PER**: 일회성 이익/손실 제거 후 정상적 이익 기준 PER
- **평균PER**: 여러 해 EPS의 평균 사용 → 일시적 실적 변동 완화
- **정상PER**: 정상적인 수익력 기반 PER (그레이엄은 7~10년 평균 권장)
- 저PER이 항상 저평가를 의미하지 않음 (한전 사례: 일회성 이익으로 PER 왜곡)

### 종목 선정 기준 (그레이엄 퀀트)

**NCAV 전략** (청산가치 기반):
- NCAV = 유동자산 - 총부채 (비유동자산을 0으로 평가하는 극보수적 지표)
- 매수 조건: NCAV > 시가총액 (완화) 또는 NCAV > 시가총액 × 1.5 (원칙)
- 한국 백테스트 연복리 ~25-30%

**그레이엄의 마지막 선물** (1976):
- PER 10배 이하 (한국은 5배 이하가 더 효과적)
- 부채비율 50% 이하
- 한국 백테스트 연복리 ~14-16%

**업그레이드 버전**:
- ROA 5% 이상 + 부채비율 50% 이하 + PBR 낮은 순 20-30종목
- PBR 0.2배 이하 제외
- 한국 백테스트 연복리 ~18%

### 투자 원칙 요약
- 우량기업(ROE > r)을 적정 가격에 매수 → 시간이 갈수록 내재가치 상승 ("시간은 내 친구")
- 불량기업(ROE < r)이라도 내재가치 대비 극도로 저평가 시 → 예외적 투자 가능 (전문가 영역)
- ROE < r 이면서 내재가치보다 주가도 비싼 경우 → "지옥행 특급열차" (이 영역만 피해도 절반 성공)
